# 📦 Product Dashboard & Form Validation App

A modern **Next.js frontend project** demonstrating real-world development skills including API integration, infinite scrolling, and form validation using industry-standard tools.

---

## 🚀 Features

### 📊 Product Dashboard (Task 1)

- API-based product listing
- Infinite scroll using React Query
- Search functionality with debounce
- Sorting (price & name)
- Responsive and clean grid UI
- Loading & error states handling

### 🧾 Form Validation (Task 2)

- Built with Formik + Yup
- Full validation (name, email, phone, password)
- Inline error messages
- Success feedback on submission
- Clean and responsive UI

---

## 🛠 Tech Stack

Next.js • React • TypeScript • React Query • Formik • Yup • Tailwind CSS

---

## 🔗 Routes

- `/` → Product Dashboard
- `/user-form` → Form Validation
