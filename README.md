📦 Product Dashboard & Form Validation App

A modern Next.js frontend project demonstrating real-world development skills including API integration, infinite scrolling, and form validation.

🚀 Features
📊 Product Dashboard (Task 1)
API-based product listing
Infinite scroll (React Query)
Search with debounce
Sorting (price & name)
Responsive UI with clean design
🧾 Form Validation (Task 2)
Formik + Yup validation
Full input validation (name, email, phone, password)
Inline error handling
Success message on submit

🛠 Tech Stack
Next.js • React • TypeScript • React Query • Formik • Yup • Tailwind CSS

🔗 Routes
/ → Product Dashboard
/user-form → Form Validation

⚙️ Getting Started
1️⃣ Install dependencies
npm install
2️⃣ Run development server
npm run dev
3️⃣ Open in browser
http://localhost:3000
